== Requirements

To build and run this project, you need the following installed on your machine:

* **Java 25**: The project is configured for Java 25.
* **Maven**: Version 3.9.11 or compatible.
* **Node.js**: Version v24.11.1 or compatible.
* **npm**: Version 11.6.2 or compatible is recommended.

You can verify your installed versions with the following commands:

[source,bash]
----
java -version
mvn -v
node -v
npm -v
----

== Running the project in development mode

The project consists of a server (Spring Boot) and a client (Angular).

=== Server

The server is a Spring Boot application. To run it in development mode:

. Run the application from the project root:
+
[source,bash]
----
mvn -f server/pom.xml spring-boot:test-run -am -pl boat-manager-application
----

The server will start on `http://localhost:8080`.

=== Client

The client is an Angular application.

. Install dependencies (if not already done):
+
[source,bash]
----
npm --prefix client install
----

. Start the development server:
+
[source,bash]
----
npm --prefix client start
----

The client will be available at `http://localhost:4200/`.
